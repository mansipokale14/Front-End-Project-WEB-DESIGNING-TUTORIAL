Web Terminology

Web terminology refers to the essential terms and concepts used in web development and internet technologies. These include the languages for building web pages, like HTML, CSS, and JavaScript, as well as protocols like HTTP and HTTPS that govern data transfer. It also encompasses concepts such as frontend and backend development, APIs for software communication, and the role of web servers and databases. Understanding these terms is crucial for navigating the processes involved in creating, managing, and interacting with websites and web applications.

Introduction Of Web Terminology

  2. Key components Of Web Terminology
    1. HTML (Hypertext Markup Language): The standard language used to create and structure web pages.
    2. CSS (Cascading Style Sheets): The language used to control the visual appearance and layout of web pages.
    3. JavaScript: A programming language that adds interactivity and dynamic content to websites.
    4. HTTP/HTTPS (Hypertext Transfer Protocol/Secure): Protocols used for transferring data over the web, with HTTPS providing a secure, encrypted connection.
    5. URLs (Uniform Resource Locators): The addresses used to access resources on the web.
    6. Frontend: The part of a website that users interact with directly, including design, layout, and user experience.
    7. Backend: The server-side of a website, including databases, servers, and application logic that handle data processing and storage.
    8. APIs (Application Programming Interfaces): Sets of rules that allow different software applications to communicate with each other.
    9. DNS (Domain Name System): A system that translates domain names into IP addresses, allowing browsers to locate and load websites.
    10. SEO (Search Engine Optimization): Techniques used to improve a website's visibility and ranking in search engine results.
    11. Responsive Design: An approach to web design that ensures websites function well across a variety of devices and screen sizes.
  3. Web Terminology Language

    5. Task Runners: Automate tasks (e.g., Webpack).
    6. CMS: Manage content without coding (e.g., WordPress).
    7. Package Managers: Handle dependencies (e.g., npm).
    8. APIs: Enable software communication.
    9. Databases: Store data (e.g., MySQL).
  5. Career Opportunities in Web Desining
    • Web Designer
      1. Role: Focuses on the visual aspects of a website, including layout, color schemes, typography, and overall aesthetics.
      2. Skills Needed: HTML, CSS, Adobe Creative Suite (Photoshop, Illustrator), UX/UI principles.
    • UI/UX Designer
      1. Role: UI (User Interface) designers focus on the look and feel of the website, while UX (User Experience) designers focus on the overall user journey, ensuring the site is easy and enjoyable to use.
      2. Skills Needed: Wireframing, prototyping, user research, design tools like Sketch, Figma, and Adobe XD.
    • Front-End Developer
      1. Role: Bridges the gap between design and technology, implementing the design into functional websites using HTML, CSS, and JavaScript.
      2. Skills Needed: HTML, CSS, JavaScript, responsive design, frameworks like React or Angular.
    • Web Developer
      1. Role: Focuses on building and maintaining the technical aspects of a website, including both front-end and back-end development.
      2. Skills Needed: HTML, CSS, JavaScript, server-side languages (e.g., PHP, Python), databases (e.g., MySQL).

Evolution Of Computing

  • Computing started by using stand-alone computers to perform different computing operations.
  • Later organizations began to connect their computers to share data.
  • “The Advanced Research Projects Agency Network (ARPANET) was the first wide-area packet-switched network with distributed control and one of the first computer networks to implement the TCP/IP protocol suite. Both technologies became the technical foundation of the Internet.

Different Type Of Network

  • Local Area Network(LAN): Network in a small geographical area.
  • Metropolitan Area Network(MAN):Network that covers city
  • Wide Area Network(WAN): Networks that connects LAN’s and MAN’s across the globe.

Web and Internet

  • WAN raised the need to share data across the globe rather than within an organization
  • This resulted in the evolution of Web also known as World Wide Web(WWW).
  • Internet is known as the largest WAN.
  • Web is a way to access information using internet.

Important Terms:

  • Network: A collection of interconnected devices, such as computers, printers, and servers, that can communicate with each other.
  • Protocol: A set of rules and standards that define how devices on a network connected to communicate and share information among them
  • IP Address: A unique numerical identifier assigned to each device on a network, used to identify and communicate with other devices. (try out some cmd commands : ipconfig; ping chatgpt.com)
  • ISP: An ISP (internet service provider) is a company that provides individuals and organizations access to the internet and other related services. They also provide additional services such as email, online security, and home phone.
  • Firewall: A security device or software that monitors and controls incoming and outgoing network traffic, based on a set of predefined security rules.
  • DNS (Domain Name System): A system that translates domain names (such as www.example.com) into IP addresses, allowing devices to locate and connect to websites and other network resources
  • DHCP (Dynamic Host Configuration Protocol): A protocol that automatically assigns IP addresses and network configuration settings to devices on a network
  • TCP/IP (Transmission Control Protocol/Internet Protocol): A set of protocols used to communicate over the internet and other networks. Mainly used for sending data over the internet.

    Firewall

    A firewall is a crucial component of network security designed to protect systems from unauthorized access and cyber threats. It functions as a barrier between a trusted internal network and an untrusted external network, such as the internet. By monitoring and controlling incoming and outgoing traffic based on predefined security rules, firewalls prevent malicious activity and unauthorized access to sensitive data. They come in various forms, including hardware firewalls, which are physical devices positioned between a network and its external connections, and software firewalls, which are programs installed on individual devices to manage traffic for that specific machine. Additionally, application firewalls focus on protecting specific applications from attacks, while next-generation firewalls offer advanced features such as intrusion prevention and deep packet inspection. Overall, firewalls are essential for safeguarding networks by filtering traffic, enforcing access controls, and providing monitoring and logging capabilities to ensure the security and integrity of digital systems.
