Introduction Of JavaScript

JavaScript is a versatile, high-level programming language primarily used for adding interactivity, functionality, and dynamic behavior to web pages. It is one of the core technologies of web development, alongside HTML and CSS. JavaScript enables developers to create engaging user experiences by allowing web pages to respond to user actions, validate forms, update content dynamically, and much more.

  1. JavaScript Basics
    1. Syntax and Structure
      • Variables: Used to store data values. 
                            var name = "John";  // Old way of declaring variables
                        let age = 30;       // Preferred way in modern JS
                        const pi = 3.14;    // Constants cannot be reassigned
      • Data Types: JavaScript has several built-in data types, including:
        • Primitive Types: String, Number, Boolean, Null, Undefined, Symbol, BigInt
        • Complex Types: Object, Array, Function
      • Operators: JavaScript supports various operators for performing arithmetic (+, -, *, /), comparison (==, ===, !=, >, <), logical (&&, ||, !), and other operations.
      • Control Structures: JavaScript uses control structures like if, else, for, while, switch, etc., to control the flow of execution. 
                            if (age >= 18) {
                        console.log("You are an adult.");
                    } else {
                        console.log("You are a minor.");
                    }
      • Functions: Functions are blocks of code designed to perform particular tasks. They are defined using the function keyword or using arrow function syntax in modern JavaScript. 
                            function greet(name) {
                        return "Hello, " + name + "!";
                    }
                    
                    const greet = (name) => "Hello, " + name + "!";
      • Scoping and Hoisting
        • Scope: Refers to the visibility of variables. JavaScript has function scope, block scope (introduced with let and const), and global scope.
        • Examole 
                          function example() {
                    var x = 10;  // Function-scoped
                    if (true) {
                        let y = 20;  // Block-scoped
                    }
                    console.log(x);  // 10
                    // console.log(y);  // Error: y is not defined
                }
        • function example() { var x = 10; // Function-scoped if (true) { let y = 20; // Block-scoped } console.log(x); // 10 // console.log(y); // Error: y is not defined }
        ```

        Working with Objects and Arrays

        1. Objects Objects are collections of key-value pairs, where keys (also called properties) are strings, and values can be any data type. 
                              const person = {
                        firstName: "John",
                        lastName: "Doe",
                        age: 30,
                        greet: function() {
                            return "Hello, " + this.firstName + "!";
                        }
                    };
                    
                    console.log(person.firstName);  // John
                    console.log(person.greet());    // Hello, John!
        2. Arrays Arrays are ordered collections of items. Items in an array can be of any data type, and arrays themselves are a type of object.

        JavaScript Events and DOM Manipulation

        1. a. Events JavaScript interacts with the HTML content of a webpage through events. Events are actions or occurrences that happen in the browser, such as clicking a button, submitting a form, or pressing a key.
        2. DOM (Document Object Model) Manipulation The DOM is a programming interface for HTML and XML documents. It represents the document as a tree structure where each node is an object representing a part of the document.
          • Selecting Elements: Elements can be selected using methods like getElementById(), getElementsByClassName(), querySelector(), etc.
          • Modifying Elements: JavaScript allows you to modify the content, style, and attributes of elements in the DOM.
          • Event Propagation: JavaScript handles event propagation through capturing and bubbling phases, allowing for more complex interactions like event delegation.

        Asynchronous JavaScript

        1. Callbacks A callback is a function passed into another function as an argument, which is then executed inside the outer function to complete some kind of routine or action.
        2. Promises Promises represent the eventual completion (or failure) of an asynchronous operation and its resulting value. A promise can be in one of three states: pending, fulfilled, or rejected. 
                                  const promise = new Promise((resolve, reject) => {
                            const success = true;  // Simulate success/failure
                            if (success) {
                                resolve("Data fetched successfully");
                            } else {
                                reject("Error fetching data");
                            }
                        });
                        
                        promise.then((message) => {
                            console.log(message);
                        }).catch((error) => {
                            console.error(error);
                        });
        3. Async/Await async and await are modern JavaScript features that allow you to write asynchronous code that looks and behaves like synchronous code, making it easier to read and understand. 
                                  async function fetchData() {
                            try {
                                const response = await fetch("https://api.example.com/data");
                                const data = await response.json();
                                console.log(data);
                            } catch (error) {
                                console.error("Error:", error);
                            }
                        }
                        
                        fetchData();

        JavaScript Modules

        1. ES6 Modules ES6 introduced a standardized module system that allows you to export and import functions, objects, or primitives between different JavaScript files.
          • Exporting: 
                                            // math.js
                                    export const pi = 3.14;
                                    export function add(a, b) {
                                        return a + b;
                                    }
          • Importing: 
                                            // main.js
                                    import { pi, add } from './math.js';
                                    console.log(pi);  // 3.14
                                    console.log(add(2, 3));  // 5

        JavaScript in the Browser vs. JavaScript on the Server

        1. Client-Side JavaScript Client-side JavaScript is executed in the user's browser, allowing developers to create interactive and dynamic web pages. Tasks: DOM manipulation, event handling, form validation, AJAX requests, etc.
        2. Server-Side JavaScript With the advent of Node.js, JavaScript can also be used on the server side
